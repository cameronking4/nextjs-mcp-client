Testing the Open MCP Chat Widget

This page demonstrates how the Open MCP chat widget can be embedded in any website. The widget is displayed in the sidebar and allows users to interact with the AI assistant.

Features

  • Responsive design that works on all devices
  • Customizable appearance
  • Access to selected MCP servers
  • Domain restrictions for security
